About me

Hello! I have been actively learning how to code in my spare time since November of 2016.

My coding journey began with Codecademy, where I learned HTML/CSS. From there I found Free Code Camp, where I earned the Front End Development Certification. I took a few classes from Code School, then completed Colt Steele’s Web Developer Bootcamp on Udemy, where I built a full-service web application utilizing Node.js, Express.js, Passport.js, MongoDB, and deployed it with Heroku.

Coding is AWESOME! I love learning new technologies and creating sweet projects. When I am in the zone, I am so focused on solving the problems and getting the job done that everything else falls away. It’s pure flow!

I am eagerly looking for a position as a Developer with a great company. Take me on, and together we will build many amazing things!
